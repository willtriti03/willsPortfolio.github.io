게임 소개

어두운 밤에 길을 걷다가 비명소리에 이끌려 다가간 공사장에서 당신은 살인사건을 목격하게 된다.
살인사건 현장에 정신이 팔린 나머지 살인범이 뒤에서 다가오는 것을 알아차리지 못하고 당하고 만다.
정신을 차렸을 땐 안면인식장애가 생기고 사람들의 얼굴이 매번 다르게 보이기 시작한다!

당신은 유일한 목격자로 범인을 찾기 위해 돌아다니지만
당신의 안면인식을 이용한 범인은 당신의 주변 인물인척 교묘하게 트릭을 설치하기 시작한다.

얼굴을 기억하지 못하는 당신은 사람들과의 대화만을 기억하여 범인을 찾아낼 수 있을까!


제작 동기

게임잼의 주제가 Reset이었는데,
그에 맞춰 로그라이크 게임과 매일 매일 얼굴이 달라지는 추리 게임 중 고민을 하다가
추리게임이 좀 더 제작하는 동안 재미있을 것 같고 로그라이크 게임은 많이 나올 것 같아
NOT을 제작하게 되었습니다.


게임 장르

게임 개요

친구를 죽인 살인범이 누군지 찾아내자


게임 요소

자유도
- 맵을 자유롭게 돌아다니면서 대화 밑 증거 수집
- 특정 증거를 못 찾고 넘어가면 범인 색출이 불가하다.

플레이어의 안면인식장애를 이용한 살인범의 교묘한 트릭
- 플레이어의 안면인식을 이용하여 다른 사람인척하고 거짓 정보를 흘린다.

쓸데없는 대화는 존재하지 않음
- 플레이어의 안면인식을 살인범이 이용하기 있기 때문에 평소에 나누었던 대화를 토대로
상대가 진짜 상대가 맞는지 판별해야한다.

모든 게임 캐릭터는 안면을 가지지 않음
- 플레이어의 안면인식장애를 표현하기 위해 모든 캐릭터는 얼굴이 없다.


게임 승리 / 패배 요소

승리 요소
- 진범을 잡는다.

패배 요소
- 진범을 잡지 못해 미해결 사건이 된다.




회고

같은 팀 디자이너에 공군이면서 휴가나온 서울대 시각디자인과 형의 디자인으로 엄청나게 이쁘면서
분위기가 살아난 게임이었던 것 같다. 기획으로 참여하여서 진행했지만 시간이 촉박해지고
마지막엔 개발자가 도망가는 바람에 개발 마무리는 내가 했었던 것으로 기억에 남는다.

2박3일간 스토리 기반의 게임을 만든 다는 것은 상당히 어려웠다.
그래픽에 응하지 못하는 약간의 부족한 스토리가 아쉬웠다.


평소 추리물을 좋아하고 디트로이드 비컴 휴먼을 해보고 나선 인터렉티브 게임장르를 만들어 보고 싶어
군 생활동안 스토리를 인터렉티브 추리 게임으로 재작업하고 있다.(현재 진행 중)
